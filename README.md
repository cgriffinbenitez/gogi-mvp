# GOGI MVP

GOGI is an AI-powered, pedagogically aligned platform designed to improve reading and writing outcomes in high school classrooms.

## MVP Goal
Build a working prototype that supports:

- Reading comprehension
- Evidence-based writing
- AI-guided thinking support
- Teacher visibility into student performance

## Core Loop
teacher assigns → student reads → AI supports thinking → student writes → teacher sees mastery data
